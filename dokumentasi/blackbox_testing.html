DOKUMENTASI BLACK BOX TESTING

Aplikasi: Sistem Prediksi Nasabah BPR Bogor Jabar
Metode: Black Box Testing (Functional Testing)
Tujuan: Menguji fungsionalitas aplikasi berdasarkan spesifikasi tanpa melihat kode internal

1. Pengujian Fitur Dashboard

No Deskripsi Pengujian Hasil Pengujian Keterangan
1 Menampilkan total sesi prediksi yang telah dilakukan Sistem berhasil menampilkan jumlah total sesi prediksi dengan benar PASS
2 Menampilkan total nasabah dengan prediksi aktif Sistem berhasil menampilkan jumlah nasabah aktif dari semua sesi PASS
3 Menampilkan total nasabah dengan prediksi tidak aktif Sistem berhasil menampilkan jumlah nasabah tidak aktif dari semua sesi PASS
4 Navigasi ke halaman riwayat prediksi Tombol "Lihat Semua" berhasil membuka halaman riwayat prediksi PASS

2. Pengujian Fitur Input Data Manual

No Deskripsi Pengujian Hasil Pengujian Keterangan
1 Menampilkan form input data nasabah dengan semua field yang diperlukan Form berhasil menampilkan 9 field input (ID Nasabah, Usia, Jenis Kelamin, dll) PASS
2 Validasi field kosong saat menambah data Sistem menampilkan pesan error jika ada field yang kosong PASS
3 Validasi input angka pada field numerik Field usia, pendapatan, frekuensi, saldo, dan lama nasabah hanya menerima input angka PASS
4 Menambahkan data nasabah ke daftar sementara Data berhasil ditambahkan dan ditampilkan dalam daftar sementara PASS
5 Edit data nasabah dari daftar sementara Sistem berhasil memuat data ke form dan mengupdate setelah edit PASS
6 Hapus data nasabah dari daftar sementara Data berhasil dihapus dari daftar sementara PASS
7 Clear form setelah menambah data Form berhasil dikosongkan setelah data ditambahkan PASS
8 Submit prediksi dengan minimal 1 data nasabah Sistem berhasil memproses prediksi dan menampilkan hasilnya PASS
9 Submit prediksi tanpa data nasabah Sistem menampilkan pesan error dan tidak memproses prediksi PASS

3. Pengujian Fitur Upload Excel

No Deskripsi Pengujian Hasil Pengujian Keterangan
1 Menampilkan halaman upload dengan informasi format Excel Sistem menampilkan panduan format Excel dengan 9 kolom yang diperlukan PASS
2 Memilih file Excel dengan format .xlsx File picker hanya menampilkan file dengan ekstensi .xlsx dan .xls PASS
3 Membaca dan parsing data dari file Excel Sistem berhasil membaca data dari file Excel dan menampilkan jumlah data yang valid PASS
4 Validasi format data Excel (kolom tidak lengkap) Sistem melewati baris dengan kolom tidak lengkap dan hanya memproses data valid PASS
5 Validasi nilai Jenis Kelamin (harus Laki-laki atau Perempuan) Data dengan jenis kelamin selain Laki-laki/Perempuan tidak diproses PASS
6 Validasi nilai Status Nasabah (harus Aktif atau Tidak Aktif) Data dengan status selain Aktif/Tidak Aktif tidak diproses PASS
7 Menampilkan preview data yang berhasil dibaca Sistem menampilkan preview dengan ID nasabah, pekerjaan, dan status PASS
8 Submit prediksi dari data Excel Sistem berhasil memproses semua data Excel dan menampilkan hasil prediksi PASS
9 Clear data Excel yang sudah dipilih Tombol close berhasil menghapus file dan data Excel yang sudah dipilih PASS

4. Pengujian Fitur Detail Prediksi

No Deskripsi Pengujian Hasil Pengujian Keterangan
1 Menampilkan ringkasan hasil prediksi (jumlah data dan akurasi) Sistem menampilkan header dengan jumlah total data dan persentase akurasi PASS
2 Menampilkan summary ID nasabah yang diprediksi aktif Sistem menampilkan daftar ID nasabah dengan final prediksi "Aktif" dalam bentuk badge PASS
3 Menampilkan summary ID nasabah yang diprediksi tidak aktif Sistem menampilkan daftar ID nasabah dengan final prediksi "Tidak Aktif" dalam bentuk badge PASS
4 Filter data nasabah berdasarkan status "Semua" Sistem menampilkan semua data nasabah tanpa filter PASS
5 Filter data nasabah berdasarkan status "Aktif" Sistem hanya menampilkan nasabah dengan final prediksi "Aktif" PASS
6 Filter data nasabah berdasarkan status "Tidak Aktif" Sistem hanya menampilkan nasabah dengan final prediksi "Tidak Aktif" PASS
7 Menampilkan counter jumlah data yang sedang ditampilkan Counter menampilkan jumlah data sesuai filter yang dipilih PASS
8 Menampilkan detail setiap nasabah (prediksi pohon, final prediksi, evaluasi) Sistem menampilkan ID nasabah, prediksi awal, hasil 5 pohon, final prediksi, dan evaluasi PASS
9 Menampilkan pesan jika tidak ada data dengan filter tertentu Sistem menampilkan icon dan pesan "Tidak ada data nasabah" jika hasil filter kosong PASS

5. Pengujian Fitur Download PDF

No Deskripsi Pengujian Hasil Pengujian Keterangan
1 Generate laporan PDF dari hasil prediksi Sistem berhasil membuat file PDF dengan format laporan yang lengkap PASS
2 Menampilkan loading indicator saat generate PDF Sistem menampilkan dialog loading "Membuat laporan PDF..." selama proses PASS
3 Menampilkan dialog share setelah PDF selesai dibuat Sistem membuka dialog share untuk memilih lokasi penyimpanan atau aplikasi share PASS
4 Format laporan PDF mencakup header dan informasi sesi PDF berisi header "BPR Bogor Jabar", session ID, tanggal prediksi PASS
5 Format laporan PDF mencakup ringkasan hasil PDF menampilkan total data, prediksi aktif/tidak aktif, dan akurasi PASS
6 Format laporan PDF mencakup detail setiap nasabah PDF menampilkan detail lengkap untuk setiap nasabah termasuk prediksi pohon PASS
7 Menampilkan notifikasi sukses setelah PDF tersimpan Sistem menampilkan snackbar "Laporan PDF siap dibagikan" dengan icon success PASS

6. Pengujian Fitur Riwayat Prediksi

No Deskripsi Pengujian Hasil Pengujian Keterangan
1 Menampilkan daftar semua riwayat prediksi Sistem menampilkan semua sesi prediksi yang pernah dilakukan PASS
2 Menampilkan informasi setiap riwayat (tanggal, jumlah data, akurasi) Setiap item riwayat menampilkan tanggal, jumlah nasabah, dan persentase akurasi PASS
3 Membuka detail prediksi dari riwayat Klik pada item riwayat berhasil membuka halaman detail prediksi PASS
4 Menghapus riwayat prediksi Tombol hapus berhasil menghapus data dari database dan refresh daftar PASS
5 Menampilkan pesan jika belum ada riwayat Sistem menampilkan pesan "Belum ada riwayat prediksi" jika database kosong PASS

7. Pengujian Navigasi Aplikasi

No Deskripsi Pengujian Hasil Pengujian Keterangan
1 Bottom navigation menampilkan 3 menu (Home, Tambah, Riwayat) Bottom navigation bar menampilkan 3 icon dengan label yang sesuai PASS
2 Navigasi ke halaman Dashboard (Home) Klik icon Home berhasil menampilkan halaman dashboard PASS
3 Navigasi ke halaman Pilihan Input Klik icon Tambah (tengah) menampilkan halaman pilihan metode input PASS
4 Navigasi ke halaman Riwayat Prediksi Klik icon Riwayat berhasil menampilkan halaman riwayat prediksi PASS
5 Back button pada detail prediksi Tombol back berhasil kembali ke halaman sebelumnya PASS
6 Highlight menu yang sedang aktif Menu yang sedang dibuka ditandai dengan warna yang berbeda PASS
